Detailed Description

Interface for type that is aware of parameters for evapotranspiration in certain calculations, and can accept such parameters being provided.

It is not necessarily the case that implementations use ET params in any way, only that they provide an interface for accepting them. As such, it is not necessarily the case that implementations maintain any provided params as part of their state. The details of how this is handled should be documented for individual subtypes.
